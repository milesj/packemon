Files

The files command will list all files within a package that will be distributed, based on the files property in package.json (if declared), and taking into account any ignore files. This is useful for verifying the contents of a package is what you expect before publishing.

packemon files <package name>

Files supports the following command line options.

  • --format - The format to display the files in. Accepts "tree" (default) or "list".
